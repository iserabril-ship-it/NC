Conectá la plataforma con Google Sheets

☁ Paso 1 — Crear el Google Sheet y el Apps Script

Seguí estos pasos una sola vez.

  1. Abrí sheets.google.com y creá una hoja nueva llamada "NC Centro Estant".
  2. En esa hoja, hacé clic en Extensiones → Apps Script.
  3. Borrá todo el código y pegá el contenido del archivo PASO1_AppScript_Backend_v2.gs.
  4. Guardá el proyecto (Ctrl+S). Nombre: "NC Backend".
  5. Hacé clic en Implementar → Nueva implementación → App web.
  6. Ejecutar como: Yo. Acceso: Cualquier persona. Implementar.
  7. Autorizá los permisos y copiá la URL generada.

🔗 Paso 2 — Conectar con esta plataforma

Pegá la URL del Apps Script y hacé clic en Conectar.

📥 Paso 3 (opcional) — Importar NC históricas

Importá las NC N°2 a N°16 que ya tenías cargadas.
